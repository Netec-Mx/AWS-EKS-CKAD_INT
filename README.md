# AWS-EKS-CKAD_INT
Este curso proporciona una visión avanzada y práctica sobre la administración de Amazon EKS en entornos productivos. Se enfoca en resiliencia, confiabilidad, seguridad y sustentabilidad, utilizando herramientas de última generación como ArgoCD, Karpenter, Istio, CrossPlane y Kepler.
